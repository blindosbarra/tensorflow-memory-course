# NumPy, pensare per array

> **La lezione si segue nel notebook** `notebooks/lezione-06-numpy.ipynb`.
> Questa pagina e' il riassunto di riferimento. Con questa lezione comincia
> la Fase 0 (fondamenti numerici), erogata prima della Fase 2 come da
> syllabus.

## Cosa saprai fare

Ragionare per array invece che per cicli: vettorizzazione, riduzioni con
`axis`, maschere booleane e slicing — il modello mentale condiviso da NumPy,
pandas e TensorFlow.

## Nel progetto

La matrice di feature della Lezione 5 viene analizzata come array: medie
delle feature per classe con maschere e riduzioni, per verificare che i tipi
di memoria abbiano firme diverse (il segnale che un classificatore potra'
sfruttare).
