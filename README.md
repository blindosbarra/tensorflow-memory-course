# TensorFlow Memory AI Course Factory

Questo repository starter contiene la specifica e l’harness necessari per far costruire a Codex un corso completo e verificabile.

## Avvio

1. Crea un repository GitHub e copia questi file.
2. Collega il repository a Codex.
3. Apri un thread Codex.
4. Incolla il prompt “Prompt di avvio per Codex” contenuto in `COURSE_FACTORY_SPEC.md`.
5. Fai completare prima la vertical slice `data-cleaning-01-missing-values`.
6. Provala personalmente.
7. Solo dopo assegna gli altri moduli.

## Documento principale

Leggi [`COURSE_FACTORY_SPEC.md`](COURSE_FACTORY_SPEC.md).

## Obiettivo

Il risultato finale è un sito statico, un insieme di notebook eseguibili e un progetto Memory AI completo, dal dataset al deploy e monitoring.
